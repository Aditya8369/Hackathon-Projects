## 🚀 Why This Project

Pollution Control Hub helps users:

- Track air quality in real time
- Decode pollutant impact on health
- Discover local hotspots visually on a map
- Participate through community-powered reports

## ✨ Key Features

- 📊 Real-time AQI dashboard using Open-Meteo Air Quality API
- 🧪 Pollutant tracking: PM2.5, PM10, CO, NO2, Ozone
- 🏙️ City-wise AQI comparison with charts
- 📍 Geolocation-based tracking of nearby pollution zones
- 🗺️ Interactive Leaflet map with hotspot view
- 🔔 Pollution alerts with browser notifications
- 🩺 Health advisory content with prevention tips
- 🌱 Solutions and awareness section with policy references
- 🤝 Community reporting with image upload + upvoting (localStorage)
- 📈 Weekly/monthly insights + simple AQI prediction trend

## 🧰 Tech Stack

- React + Vite
- Recharts (visual analytics)
- React Leaflet (maps)
- Open-Meteo Air Quality API

## 🖥️ Run Locally

```bash
npm install
npm run dev
```

Default app URL: http://localhost:5173

## 📌 Notes

- Allowing location access improves map and tracking relevance.
- Browser notifications require user permission.
- Community reports are stored locally in the browser.

## 🌟 Vision

From awareness to action, Pollution Control Hub aims to make environmental data understandable, actionable, and community-driven.
